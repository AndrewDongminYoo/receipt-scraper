# Suggested commands

- `yarn start`: start Metro.
- `yarn ios`: build and launch the iOS app.
- `yarn android`: build and launch the Android app.
- `yarn lint`: run ESLint across the repo.
- `yarn test`: run Jest tests.
- `bundle install`: install CocoaPods Ruby dependencies.
- `cd ios && bundle exec pod install`: install iOS pods after native dependency changes.
- `npx react-native doctor`: inspect React Native environment health.
- `xcrun simctl list devices available`: list available iOS simulators on macOS.
- `emulator -list-avds`: list available Android virtual devices.
- General Darwin utilities commonly used here: `git`, `ls`, `find`, `rg`, `sed`, `pwd`, `cd`.
